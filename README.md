# praktika_fikl_2023
1. Конвертировать pdf в txt (в интернете или с помощью pdf2txt.py)
2. Подготовить файл (стереть всё оглавление и оставить только содержимое глав)
3. Отредактировать regex в скрипте (в зависимости от того, как в книге выглядит нумерация)
4. "Please enter a substituted string:" — удаляет содержимое колонтитула, если есть. Если нет, то оставить пустым
5. Запустить скрипт, всё должно получиться

Скрипт использует этот словарь для сверки: https://gitlab.com/strepon/czech-cc0-dictionaries
Словарь обработан скриптом dict.py
